<br>


## 📝 앱 소개
소확행이란? '소소하지만 확실한 행복'

행복은 모든 사람이 추구하는 감정이자 삶의 목적이에요.<br>
평범한 하루에도 작고 사소하지만 나 자신에게 행복을 가져다준 순간은 항상 존재해요.<br>
하루하루 흘러가는 일상 속에서 지나쳤던 나의 소중한 행복을 찾아나가는 여정을 함께 시작해보아요😊

<br>

## 🕹️ 기능

<br>

### 1. 소셜 로그인
<img width="1363" alt="sosohappy readme 001" src="https://github.com/So-So-Happy/SoSoHappy-iOS/assets/82339184/01ef008f-c7bb-47c4-8f76-390686fec35a">
<br>
<br>

- 구글, 카카오, 애플 소셜로그인을 지원해요.
- 프로필을 설정해서 서비스를 사용할 수 있어요.

<br>

### 2. 소소해피 캘린더
<img width="1363" alt="sosohappy readme 002" src="https://github.com/So-So-Happy/SoSoHappy-iOS/assets/82339184/f10f4c45-8647-46b9-853f-ab8d1dc80a77">
<br>
<br>

- 오늘의 소확행을 기록하며, 소소해피만의 귀여운 캐릭터로 나만의 소소해피 캘린더를 완성해보세요.
- 다양한 카테고리로 기록한 소확행들을 확인할 수 있어요.

<br>

### 3. 소소해피 일지 작성
<img width="1363" alt="sosohappy readme 003" src="https://github.com/So-So-Happy/SoSoHappy-iOS/assets/82339184/c77f9292-35c2-473c-a241-87af1981fbc7">
<br>
<br>

- 오늘의 날씨, 행복지수, 나에게 행복을 주었던 카테고리와 함께 행복의 순간을 기록해보세요.
- 기록한 행복 일지를 피드에서 다른 사용자들과 공유할 수 있어요.
- 기록한 행복 일지에 자물쇠를 잠그면 나만 볼 수 있어요.

<br>

### 4. 소소해피 통계
<img width="1363" alt="sosohappy readme 004" src="https://github.com/So-So-Happy/SoSoHappy-iOS/assets/82339184/91ffeb4e-0039-468c-bf31-c645ec8fb486">
<br>
<br>

- 이번 달에는 어떤 카테고리에서 행복을 가장 많이 느꼈는지 순위를 매겨 드릴게요.
- 기록한 행복 카테고리를 바탕으로 좋아하실만한 소확행을 추천해드려요.
- 행복 지수를 그래프로 정리하여 한눈에 행복 지수 변화 추이를 볼 수 있어요.
  
<br>

### 5. 소소해피 피드
<img width="1363" alt="sosohappy readme 005" src="https://github.com/So-So-Happy/SoSoHappy-iOS/assets/82339184/6c20e82c-59a7-4f1b-8041-2d799489c0d0">
<br>
<br>

- 이웃들의 소소해피를 확인할 수 있어요.
- 이웃들의 프로필을 통해 이웃들이 작성한 소소해피 기록들을 확인할 수 있어요.
  
<br>

### 6. 소소해피 응원
<img width="1363" alt="sosohappy readme 006" src="https://github.com/So-So-Happy/SoSoHappy-iOS/assets/82339184/8eef6e68-d021-4606-834b-f16b50f32b3f">
<br>
<br>

- 이웃이 작성한 행복 일지에 하트를 눌러 응원 알림을 보내보세요.
  
<br>

### 7. 마이페이지
<img width="1363" alt="sosohappy readme 007" src="https://github.com/So-So-Happy/SoSoHappy-iOS/assets/82339184/3f15a25f-1b5f-4b51-959e-ba78a0dca476">
<br>
<br>

- 프로필 수정과 알림설정을 할 수 있어요.
- 로그아웃 및 회원탈퇴를 할 수 있어요.

<summary> 커밋 타입 </summary>
<div markdown="1">   
  
```
- [Feat] 새로운 기능 추가
- [Chore] 코드 수정, 내부 파일 수정, 주석
- [Add] Feat 이외의 부수적인 코드 추가, 파일추가, 라이브러리 추가, 에셋 추가
- [Style] UI 작업 → 컴포넌트 추가, xib 작업
- [Fix] 버그 해결
- [Del] 파일 삭제
- [Move] 파일 이름 / 위치 변경
- [Refactor] 코드 리팩토링
```
